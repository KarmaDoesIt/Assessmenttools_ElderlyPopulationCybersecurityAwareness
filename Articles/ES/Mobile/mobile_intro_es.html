¿Qué es el malware móvil?

Malware móvil es un software malicioso que está diseñado para infectar y dañar dispositivos móviles como teléfonos o tabletas. Así como los virus y otros tipos de malware que amenazan las computadoras, el malware móvil puede robar información personal, dañar su dispositivo e interrumpir operaciones normales. Este tipo de amenaza se puede distribuir por medio de aplicaciones maliciosas, sitios web infectados o enlaces en los que se hace clic sin saberlo.
